Dasar Bahasa Pemrograman: HTML

Posted by Marceila on Monday, June 21st 2027

Pembuatan aplikasi web secara modern menggunakan tiga fundamental technologies: HTML, CSS and JavaScript.

Dalam kuliah ini, mari fokus pada HTML. Kita akan belajar tentang HTML, bagaimana kita bisa menggunakannya.

Apa itu HTML?

HTML terdiri dari HyperText Markup Language. HTML merupakan markup language yang dipergunakan developer untuk membuat struktur dan mendefinisikan konten website.

HTML terdiri elements yang mendefinisikan berbagai macam konten seperti paragraphs, links, headings, images, video, dan lainnya. Web browsers akan memahami konten tersebut dan melakukan render dalam sebuah website.

Dalam HTML, tiap element dibagi menjadi tiga bagian:

  1. The opening tag
  2. The closing tag
  3. The actual element

Anda bisa belajar lebih jauh di MDN Web Docs.
